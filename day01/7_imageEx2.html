The picture Element

반응형 웹 (responsive web)이란 하나의 웹사이트로 데스크탑 pc, 스마트폰 태블릿 pc 등 접속하는 디스플레이의 종류에 따라 화면의 크기가 자동으로 변하도록 만든 웹 페이즈를 말합니다.
위의 picture 태그는 화면 크기에 따라 보이는 이미지를 다르게 합니다. 화면크기(viewport)가 650픽셀보다 이상은 첫벌째 이미지, 465보다크고 650보다 작을때는 두번째 이미지, 465 이하일때는 세번째 이미지입니다.

The img element is required as the last child tag of the picture declaration block. The img element is used to provide backward compatibility for browsers that do not support the picture element, or if none of the source tags matched.

개발자 도구에서 viewport 크기를 조절하면서도 실행해 보세요.
